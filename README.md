#  Mental Health Journal App

A simple and interactive Java-based console application designed to assist users in tracking and improving their mental wellness. This app allows users to log their thoughts, track mood patterns, get recommendations, and more‚Äîall while demonstrating core Object-Oriented Programming concepts.

---

## Features

- User Authentication**: Register and log in to maintain private journal data.
- Mental Health Journal**: Record daily entries.
- Mood Analysis**: Analyze mood based on journal entries and get recommendations.
- Affirmation Service**: Uplifting messages to motivate users.
- Goal Recommender**: Suggests goals based on activity and mood.
- Personality Test Service**: Fun personality insights based on user responses.
- Reflection Prompt**: Inspiring questions for daily reflection.
- Sleep Tracker**: Helps track your sleep patterns.
- History**: View past mood trends and journal entries.

---

##  Tech Stack

- Language**: Java (Core)
- Approach**: Object-Oriented Programming
- Concepts Used**:
  - Classes and Objects
  - Encapsulation and Abstraction
  - Interfaces and Services
  - Collections (`Map`, `List`)
  - Exception Handling
  - Modular Code with Packages
